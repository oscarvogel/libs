Tutorial
========

Esta sección contiene un tutorial paso a paso con ejemplos para el uso de las herramientas de **arcovid19**.

En la parte 1 del tutorial, explicamos la carga de datos usando arcovid19 y algunos de los métodos y atributos heredados
de la clase DataFrame. La Parte 2 muestra la implementación de los métodos de la clase *Cases Frame* sobre la base de datos.
Por último, en la parte 3 de este tutorial, compartimos la visualización de la base de datos COVID-19 utilizando
las herramientas proporcionadas por la clase *Plot Cases* de arcovid19.
